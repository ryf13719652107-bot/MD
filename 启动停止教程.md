## 启动机器人

### 方法一：手动启动（推荐）

#### 步骤 1：启动后端服务器

打开 **PowerShell** 或 **终端**，执行：

```powershell
cd C:\Users\86137\Desktop\智能对冲马丁\backend
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000
```



#### 步骤 2：启动前端界面

**新开一个终端窗口**，执行：

```powershell
cd C:\Users\86137\Desktop\智能对冲马丁\frontend
npm run dev
```







如果终端卡死或无法响应，打开新终端执行：

```powershell
# 停止所有 Python 进程（后端）
taskkill /f /im python.exe

# 停止所有 Node 进程（前端）
taskkill /f /im node.exe
```

## 首次运行配置

### 1. 配置 API 账户

1. 访问 <http://localhost:5173>
2. 点击左侧菜单 **系统设置**
3. 填写账户信息：
   - 账户名称（如：主网账户）
   - API Key（币安 API Key）
   - API Secret（币安 API Secret）
   - 勾选/取消 **测试网**（根据你的 API 类型）
4. 点击 **保存**

### 2. 创建交易策略

1. 点击左侧菜单 **策略管理**
2. 点击 **新增策略**
3. 配置策略参数：
   - 策略名称
   - 方向（做多/做空）
   - 交易对（或从选币池自动选择）
   - RSI 参数
   - 马丁格尔参数
4. 点击 **保存**

### 3. 启动交易

1. 在策略卡片上点击 **启动** 按钮
2. 或在状态栏点击 **运行中/已停止** 开关启动主控

## 常见问题

### Q1: 端口 8000 被占用

**错误信息**：

```
[WinError 10048] 通常每个套接字地址(协议/网络地址/端口)只允许使用一次
```

**解决方法**：

```powershell
# 查找占用 8000 端口的进程
netstat -ano | findstr :8000

# 停止该进程（将 xxxx 替换为实际的 PID）
taskkill /PID xxxx /F
```

### Q2: 前端无法连接后端

**现象**：页面显示 "未连接"

**解决方法**：

1. 确认后端已启动（看到 `Uvicorn running` 信息）
2. 检查防火墙是否阻止了 8000 端口
3. 刷新浏览器页面（F5）

### Q3: 余额获取失败

**可能原因**：

- API Key 无效
- 选择的是测试网但使用了主网 API Key
- 网络连接问题

**解决方法**：

1. 检查 API Key 是否正确
2. 在状态栏切换账户（如果有多个账户）
3. 检查 `.env` 文件中的 `BINANCE_TESTNET` 设置

### Q4: 如何切换测试网/主网

**方法一**：在系统设置中添加不同账户

- 测试网账户：勾选"测试网"选项
- 主网账户：不勾选"测试网"选项

**方法二**：修改全局配置
编辑 `backend/.env` 文件：

```
# 使用测试网
BINANCE_TESTNET=true

# 使用主网
BINANCE_TESTNET=false
```
